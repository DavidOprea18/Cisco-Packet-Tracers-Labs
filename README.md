# Cisco-Packet-Tracers-Labs

This repository contains a collection of network design and configuration projects developed using **Cisco Packet Tracer**. These labs demonstrate my practical understanding of networking fundamentals, routing protocols, and security implementation, serving as a foundation for my transition into **Cybersecurity**.

## 🚀 Key Technologies & Protocols
* **Routing:** OSPF (Single-area), RIPv2, Static Routing, Inter-VLAN Routing (Router-on-a-Stick).
* **Switching:** VLAN Segmentation, 802.1Q Encapsulation, VTP, STP.
* **Services:** DHCP Server, NAT/PAT, DNS.
* **Infrastructure:** Serial Point-to-Point connections, Clock Rate synchronization, Wildcard Masking.

-----

## 📂 Featured Projects

### 1. Dynamic Routing & Redundancy (OSPF)
* **Objective:** Implementing a resilient network using OSPF to ensure fast convergence and optimal path selection.
* **Key Features:** Cost-based routing, neighbor adjacency, and area configuration.
* **Skills:** Network convergence, troubleshooting routing tables.

### 2. Network Segmentation (VLANs & DHCP)
* **Objective:** Organizing a corporate network into functional departments to improve security and broadcast management.
* **Key Features:** Multiple VLANs, 802.1Q trunking, and automated IP assignment via centralized DHCP.

### 3. Static Connectivity & WAN Simulation
* **Objective:** Establishing end-to-end connectivity between remote LANs using serial interfaces.
* **Key Features:** Static routes, Clock Rate config, and Point-to-Point link optimization.

-----

## 🛠️ How to use
1. Download and install **Cisco Packet Tracer**.
2. Clone this repository: `git clone https://github.com/numele-tau/Networking-Labs.git`
3. Open any `.pkt` file to view the topology and CLI configurations.
